# My dotfiles

This directory contains the dorfiles for my system


## Dependencies 

### Git

```
pacman -S git

```

### Stow

```
pacman -S stow
```

## Installation

```
$ git clone https://github.com/JanKaasen/dotfiles.git
$ cd dotfiles
```

then use GNU stow to create symlinks

```
$ stow .
```
